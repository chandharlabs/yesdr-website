YesDR

An AI-Powered Private Cellular Standard

YesDR Protocol Stack (YUE & YBS)

YUE - YesDR User Equipment
Layers: APP → TRN (TCP/UDP) → IP → PDCP → RLC → MAC → PHY (SDR/BPSK)

YBS - YesDR Base Station
Layers: APP → YACP → SDAP → PDCP → RLC → MAC → PHY
