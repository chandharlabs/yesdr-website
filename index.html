YesDR

An AI-Powered Private Cellular Standard

Overview

YesDR (Yes Defined Radio) is an open, AI-powered, software-defined private cellular standard designed for research, industrial, and rural deployments. It integrates advanced wireless protocol stack elements and spectrum intelligence features for seamless experimentation and deployment.

Core Components

  • YCore: YesDR Core Network managing authentication, policy, sessions, and data forwarding
  • YBS: YesDR Base Station responsible for wireless access and control
  • YUE: YesDR User Equipment acting as client or terminal
  • YSM: YesDR Spectrum Monitor for sensing, inference, and visualization
